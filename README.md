## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Prebuild the app

   ```bash
   npx expo prebuild
   ```
3. a. Build the app on Android

   ```bash
   npx expo run:android
   ```

3. b. Build the app on iOS

   install Xcode from the App Store, then run:

   ```bash  
   cd ios
   pod install
   cd ..
   npx expo run:ios
   ```
